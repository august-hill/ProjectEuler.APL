# ProjectEuler.APL

Solutions to [Project Euler](https://projecteuler.net/) problems in APL.

## Running

Using [Dyalog APL](https://www.dyalog.com/) or [GNU APL](https://www.gnu.org/software/apl/):

```bash
# Dyalog APL
dyalog -script problem_NNN/solution.apl

# GNU APL
apl -f problem_NNN/solution.apl
```
